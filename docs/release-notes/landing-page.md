# Release notes

Release notes for the latest/stable track of the Indico charm, summarizing new features,
bug fixes and backwards-incompatible changes in each revision.

For upgrading the charm, see [How to redeploy](../how-to/redeploy.md).

## Release policy and schedule

The latest/stable track of the Indico charm is updated on an as-needed basis,
for instance, for maintenance or bug fixes.
